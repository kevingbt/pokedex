# pokedex
Ceci est mon projet pokedex.

COMMENT LANCER LE PROJET ----- A EXPLIQUER 

Pour ce projet, j'ai utilisé HTML, CSS et je me suis fixé comme objectif d'utiliser uniquement du Javascript sans framework particulier car je trouvais que mes compétences sur ce langage n'étaient pas suffisantes.

Mon projet Pokedex a du coup plusieurs fonctionnalités :
- Page d'accueil : On retrouve les 20 premiers pokémons avec image, nom, id et type. On retrouve aussi une barre de recherche en temps réel, qui filtre les 20 pokemons affichés par nom. En bas de la page on a possibilité de se balader d'une page à l'autre. En cliquant sur un pokemon, on se retrouve sur la page détail de celui-ci.
- Page détail pokemon : Ici, on retrouve les informations du pokemon, d'abord ces images, de face et de dos. On retrouve à droite toutes ses stats, ie ses types, ses informations, ses statistiques, ses capacités et sa chaîne d'évolution. Il est possible en cliquant dessus, d'accéder à la page détaillé du pokemon. La page contient aussi une flèche retour menant à la page d'accueil et en bas une navigation permettant de changer de pokemon détaillé, en passant soit au pokemon suivant, soit au pokemon précédent selon leur id.

Vu que je me suis fixé de faire ce projet en JS pur, je me suis retrouvé quelque fois bloqué quant aux composants, au fait de transmettre correctement les informations des pokemons afin de les afficher, ou encore la gestion de page. Le plus bloquand a été la réflexion sur la logique métier derrière chaque aspect du projet pour que cela fonctionne. 
J'ai aussi eu des problèmes au moment de publier mon site vers GitHub Pages pour l'affichage de mes images stockées dans mon "/src". 
J'ai aussi un problème que je n'arrive pas à régler, cela conserne les derniers pokemons (entre l'id 1025 et jusqu'à la fin je crois), l'image de ces pokemons ne chargent pas, tandis que toutes les autres chargent normalement. N'ayant pas trouvé ce qui causait ce problème, je n'ai pas pu le régler.
